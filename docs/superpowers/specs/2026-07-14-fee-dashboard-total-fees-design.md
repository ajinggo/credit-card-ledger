# 手续费看板总手续费口径设计

## 目标

将手续费看板中依赖积分估值的“净收益”主指标改为可直接核对的“总手续费”，同时把月度净收益图改为月度手续费图。手续费汇总明细中的积分价值和净收益数据继续保留。

## 界面与计算

- 顶部主指标标题由“净收益 / Net Profit”改为“总手续费 / Total Fees”。
- 主指标金额等于当前手续费看板年份、月份筛选范围内记录的 `fee` 合计。
- 辅助文案显示记录数量；没有记录时显示“当前周期暂无手续费记录”。
- 图表标题由“月度净收益”改为“月度手续费”，说明改为“按月汇总实际手续费”。
- 月度柱形高度按照各月 `fee` 合计计算，点击月份后继续跳转到手续费汇总并应用月份筛选。

## 保留范围

- 不修改手续费记录的数据结构。
- 不删除积分数量、积分价值或净收益字段。
- 不修改手续费汇总表格、筛选条件、CSV 导出和净收益列。
- 平均手续费率、平均积分回报率等其他统计保持不变。

## 验证

- 静态回归测试检查看板文案和 `fee` 汇总逻辑，不允许继续使用 `pointValue - fee` 作为月度图表值。
- 使用多月份记录验证主指标筛选口径、月度柱形和值提示。
- 运行全部现有测试、语法检查、桌面和手机截图检查。
